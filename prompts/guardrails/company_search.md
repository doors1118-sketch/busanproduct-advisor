지역업체 검색 결과는 백엔드에서 부산상품 통합 플랫폼 또는 지역 조달업체 API 검색, 국세청 휴·폐업 검증, 내부 정책기업 DB와 사업자번호 기반 매칭, 개인정보 제거를 거친 구조화 데이터로 본다.

정책기업 태그는 LLM 추론값이 아니라 백엔드 매칭 결과로 취급한다.

단, 정책기업 태그는 후보 자격 정보이지 계약 가능성의 최종 판단이 아니다.
수의계약 가능 여부는 MCP로 법령·금액·계약유형·견적 방식을 검증한 뒤 판단한다.

업체 후보 표의 마지막 열에는 "계약 가능"이라고 단정하지 않는다.
"후보", "추가 확인 필요", "법적 적격성 확인 필요"로 표시한다.
