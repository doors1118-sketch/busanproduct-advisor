# 대화 맥락 — 2026-04-20 세션

## 이 세션에서 결정한 것

### 프로젝트 방향 확정
- **명칭**: 지역상생 조달 어드바이저
- **형태**: 웹 기반 인터랙티브 매뉴얼 + AI 챗봇 (HWPX 문서 매뉴얼 아님)
- **배포 채널**: 웹앱(Full UI) + 카카오톡 + 텔레그램

### 3대 핵심 기능 확정
1. **지역업체 활용 계약절차 안내** — 일반 절차가 아닌 "지역업체 우선 경로" 안내
2. **지역업체 보호제도 법적 해석** — 지역제한 가능 여부 + 법적 근거 즉시 판단
3. **지역업체 추천** — 모니터링 시스템 2만 업체 DB에서 매칭

### 기술 스택 결정
- **Gemini API** (울트라 구독과 별개. API 키 별도 발급 필요)
- **Korean Law MCP** (법제처 API 키 필요. MCP는 법제처 API의 래퍼)
- **자체 RAG** = ChromaDB에 기관 내규(L4~L6) 저장, 질문 시 벡터 검색
- **인프라**: NCP 기존 서버 1대 활용 (포트 분리), 추가 비용 0~5만원/월

### 하네스 엔지니어링 설계 완료
- 5대 구성 요소: 정체성, 도구 흐름, 출력 형식, 행동 제약, 분기 판단
- 3개 레벨: 코드 레벨(가장 강력) > 프롬프트 레벨 > 워크플로 레벨
- 코드에 검증 로직을 박아서 AI가 건너뛸 수 없게 하는 것이 핵심

### Q&A 데이터 전략
- Korean Law MCP `search_interpretations` → 해석례·질의회신 자동 수집
- Korean Law MCP `search_decisions` → 판례·감사원 결정례 자동 수집
- 사람이 직접 쓸 건 기관 고유 사례 10건 이하
