- 사이드바→메인 헤더 아래로 이동

### 3. MCP-RAG 우선순위 체계 확립
- **1순위**: MCP 도구 검색 (법제처 실시간 원문) → 가장 정확
- **2순위**: RAG 보조자료 → "참고용, MCP와 다르면 MCP 우선" 라벨
- **3순위**: 시스템 프롬프트 확정값 → fallback
- **4순위**: 사전학습 지식 → 최후 수단

### 4. 법령체계도 활용 강화
- 시행규칙 → 행정규칙(고시/예규)까지 끝까지 검색하는 규칙 추가
- "고시하는 금액" 발견 시 행정규칙 강제 검색 규칙
- chain_law_system / chain_action_basis 3단계 검색 규칙

### 5. 지역 우대 제도 → MCP 검색 행동 규칙으로 전환
- 가점제도, 적격심사, MAS 2단계 등 하드코딩 → "도구로 검색 필수" 규칙
- MAS 2단계: "납품지역 제한"이 아닌 "종합평가방식 지역가점" 교정

### 6. 계약 매뉴얼 RAG 적재
- 3개 PDF → 1,518개 청크 → ChromaDB `manuals` 컬렉션
  - 공공구매제도 실무가이드 (726청크)
  - 지방계약 실무 매뉴얼 (747청크)
  - 소기업 우선구매 안내 (45청크)

### 7. 인용 규정 다운로드 기능

- 답변 후 "📜 인용 규정 N건 다운로드" 버튼 표시

### 8. 긴 법조항 요약 원칙

- 업체 검색은 구체적 품목 언급 시에만
