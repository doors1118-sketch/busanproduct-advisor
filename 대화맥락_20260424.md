### 증상
- 질문 입력 → "📂 관련 데이터베이스 병렬 검색 중..." → 무한 대기
- 서비스 약 5분 후 `stop-sigterm` → `SIGKILL`로 강제 종료

### 원인 분석 (journalctl 로그)
| 에러 | 원인 | 해결 방법 |
|------|------|----------|
| `RAG-QA: Could not connect to tenant default_tenant` | ChromaDB 재설치로 DB 호환 깨짐 | DB 초기화 + 재적재 |
| `Vector: 'RustBindingsAPI' has no attribute 'bindings'` | ChromaDB 버전 불일치 | pip 재설치 |
| `BM25: 인덱스 없음` | `ingest_laws.py` 서버 미실행 | 서버에서 ingest 실행 |
| `RAG-MANUAL: 경로 오류` | manuals 컬렉션 서버 미적재 | `ingest_manuals.py` 실행 |
| 서비스 SIGKILL | 메모리 958.5M 피크 | timeout 증가 |

### E5 모델 로딩
